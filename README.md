Personal Expense Tracker (CLI)

A menu-driven Personal Expense Tracker developed using Python and Object-Oriented Programming.

Features :

- Add Expense
- View Expenses
- Search Expense
- Update Expense
- Delete Expense
- Expense Summary
- Automatic Expense ID Generation
- Automatic Date Generation
- CSV File Storage
- Input Validation
- Date Validation
- Menu Driven Interface

Technologies Used :

- Python 3
- Object-Oriented Programming (OOP)
- CSV Module
- Exception Handling

Project Structure :


expense.py
tracker.py
main.py
expenses.csv
README.md

Sample Menu :


1. Add Expense
2. View Expenses
3. Search Expense
4. Update Expense
5. Delete Expense
6. View Summary
7. Save and Exit
